资金揭秘

   实时成交分布图

市场之所以有涨有跌，板块之所以有热有冷，个股之所以有强有弱，都是因为一个因素的影响。这个因素，就是“资金”。资金流入市场，市场就会有机会。股市、楼市、商品无不如此资金流向哪个行业，哪个行业就会景气，个股就会上涨，机会，也就相继出现。所以，找到这个影响市场最本质的因素，把脉资金流动的规律。就能甄机会和风险，起到去伪存真的作用

   行业资金流向

如果市场出现鲜明的热点效应时，就介入热点板块个股，最好是板块的领头羊品种，往往领头羊品种是最安全的，也是涨幅最可观的品种。追涨停个股开盘过后，个股的强弱状态比较清晰，越早上攻涨停的个股，后市弹升的 机会也越多
